# Perfume_website_project
Frontend project for Premium Perfumes website


Essence is a premium perfumes frontend website created using **HTML, CSS, and JavaScript.  
This project focuses on a modern, luxury UI design with interactive features and responsive layout.

## Features

- Modern and stylish user interface
- Fixed navigation bar
- Hero section with brand introduction
- Featured perfume products
- Product category filter (For Her, For Him, for everyone)
- Add to cart functionality
- Smooth scrolling and animations
- Testimonials and newsletter section
- Fully responsive for mobile, tablet, and desktop

## Technologies Used

- HTML – Website structure  
- CSS – Styling, layout, animations, responsiveness  
- JavaScript – Interactivity, filtering, cart system
